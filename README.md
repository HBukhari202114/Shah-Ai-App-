Title: Shah-AI-Chatbot-for-Creative-Coding

Shah-AI-Chatbot-for-Creative-Coding is a next-generation AI application designed to eliminate daily friction points and unlock creative coding potential using the latest advancements of Gemini 3 Pro. By leveraging multimodal reasoning, native 3D visualization, and dynamic interaction, this chatbot transforms complex coding problems and creative ideas into actionable solutions instantly.

Whether it’s science, education, accessibility, health, business, or technology, Shah-AI adapts intelligently to any user input—text, voice, image, document, or camera feed. The AI analyzes the problem, generates a step-by-step solution, produces interactive visualizations, and provides micro-app prototypes for immediate experimentation.

Key innovations include:

Magic Mode: Automatically builds functional micro-apps from user ideas.

Global Impact Mode: Analyzes worldwide challenges with dynamic diagrams and actionable roadmaps.

Educator Mode: Explains concepts at three levels of complexity with engaging visual aids.

3D Soothing UI: Subtle animations, soft parallax depth effects, and floating particle backgrounds enhance focus while maintaining aesthetic appeal.

Accessibility Companion: Voice-only interaction, high-contrast interface, and dyslexia-friendly design ensure usability for everyone.

Shah-AI is more than a demo—it is a platform that bridges imagination and execution, showing the full potential of Gemini 3 Pro’s multimodal capabilities. Users receive instant guidance, visual storytelling, and exportable outputs like pitch decks, code snippets, and video scripts.

By combining technical sophistication, creativity, and seamless user experience, Shah-AI-Chatbot-for-Creative-Coding demonstrates a real-world, tangible impact: improving productivity, learning, decision-making, and global problem-solving—all in a single, visually captivating app.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
